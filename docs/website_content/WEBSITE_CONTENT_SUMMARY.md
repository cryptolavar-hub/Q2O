### **Tier 2: Migration Service Fees** (Quick2Odoo Billing)

**What It Is**: Per-migration charge based on data volume

**Pricing Factors**:

3. **Record count** (extra records charged per 1K)
4. **Platform complexity** (multiplier: 1.0x to 2.0x)

### **Why Both Models?**

**Subscription** = Software license + quota  
**Migration Fee** = Actual service delivery

It's like:
- **AWS**: Reserved instances + usage charges
- **Adobe**: Monthly subscription + print costs
- **Uber**: Platform fee + per-ride charges

**Benefits**:
- ‚úÖ Fair for everyone (light users pay less)
- ‚úÖ Predictable costs (subscription is fixed)
- ‚úÖ Scales with usage (heavy users pay more)
- ‚úÖ Two revenue streams (recurring + usage)
